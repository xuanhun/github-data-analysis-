For giving the easily generating SVG power to user without their token and avoiding hitting the GitHub API rate limit, we need our GitHub personal access token pool. If you like to donate one, please generate your token following the guide below.

## Guide to Generate Token

1. Sign in [GitHub](https://github.com/) with your account;
2. Go to [personal access tokens](https://github.com/settings/tokens) page;
3. Click the `Generate new token button`;

   <img width="700" src="https://user-images.githubusercontent.com/24653555/157191272-a8b5dc6d-6825-4b35-b20f-f469d3031b89.png" />

4. Give a name for token, and choose the **`No expiration`** in the Expiration seletor;

   <img width="700" src="https://user-images.githubusercontent.com/24653555/157191293-a0a08ff8-eae8-4708-a145-5db96ea993c3.png" />

5. **Do not select any scope**, just scroll down to the page footer and click Generate token button;

   <img width="700" src="https://user-images.githubusercontent.com/24653555/157191317-6df1a8b5-f0c3-4319-b8fe-ad7b9ff3d4f5.png" />

6. Click the `copy` button and paste it;

   <img width="700" src="https://user-images.githubusercontent.com/24653555/157191338-fbfa7fc4-6354-44b2-a5a6-3766daf426d8.png" />
